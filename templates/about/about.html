Sobre o Programa

Este programa é um gerador de gráficos de pizza e de barras. Permite que você insira uma lista de números separados por vírgula, gere o gráfico correspondente e o exiba em uma página.

O programa utiliza a biblioteca Matplotlib para gerar os gráficos e o framework Flask para criar a interface web. Os gráficos são salvos como arquivos de imagem (PNG) na pasta "static/img".
